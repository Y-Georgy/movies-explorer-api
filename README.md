# Бекенд movies-explorer-api

### Описание
Бекенд приложения со следующими возможностями: авторизация и регистрация пользователей, операции с карточками фильмов и с профилем пользователя.

### Стек

```
- Node.js
- Express
- JavaScript
- MongoDB
- Mongoose
- Eslint
```
